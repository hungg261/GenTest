# Changelog

All notable changes to this project will be documented in this file.  
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),  
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

Dates are formatted as **dd/mm/yyyy**.

## [1.1.0] - 16/11/2025

### Added

- `--version` flag to display the current version.
- Users can now check the version through the GUI app.
- Target folder (where test cases are generated) opens automatically after test case generation completes.

### Changed

- `-input` flag in `main.exe` is now `--input`.
- `-output` flag in `main.exe` is now `--output`.
- Some other shortened flags have been updated.
- Reformatted header files for better readability.

### Fixed

- Fixed several minor bugs.
